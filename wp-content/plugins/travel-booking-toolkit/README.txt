Travel Booking Toolkit allows you to add extra functionality to the Customizer, Widgets Section in your WordPress admin area.

== Description ==

Travel Booking Toolkit is a plugin that adds extra functionality in the Customizer and Widgets Section of your admin area to bring your website to perfection and make whole. It improves the overall performance of your website and results in better user satisfaction.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select travel-booking-toolkit.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download travel-booking-toolkit.zip
2. Extract the travel-booking-toolkit directory to your computer
3. Upload the travel-booking-toolkit directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
